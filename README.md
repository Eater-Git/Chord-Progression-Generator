# Chord-Progression-Generator

コード進行のルール(カデンツ)を基にダイアトニックコードでコード進行を生成しmidiファイルで出力します。

<!-- # DEMO -->

# 特徴
ipynb形式なのでJupyter NotebookやGoogle Colaboratoryで動作します。

# 使い方
ソースコードをクローンするかダウンロードしてJupyter NotebookやGoogle Colaboratoryで開いてください。

Google Colaboratoryで開く際はGoogleドライブにアップロードして開くほかに、Colaboratoryを立ち上げた後\[ファイル\]→\[ノートブックを開く\]から[GitHub]タブを選び、 _Eater-Git_ で検索することでこのリポジトリを開くことができます。

ノートブックを開いた後は最上部のセルを実行することで必要なライブラリをインストールします。
最下部のセルを実行することでmidiファイルを出力します。
